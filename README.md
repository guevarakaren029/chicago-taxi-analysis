# Chicago Taxi Trips Data Analysis

This project analyzes taxi trip data in Chicago to identify trends in ride demand and travel patterns.

## Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn
- Statistical hypothesis testing

## 1. Data Loading and Exploration
The datasets were loaded and explored to understand their structure.

![Data Loading](01_carga_datos.png)

## 2. Taxi Companies Analysis
Analysis of the number of trips completed by different taxi companies.

![Taxi Companies](02_analisis_empresas.png)

## 3. Top 10 Neighborhoods by Average Trips
Identification of the neighborhoods with the highest average number of completed taxi trips.

![Top Neighborhoods](03_top_barrios.png)

## 4. Hypothesis Testing
Statistical testing was performed to determine whether rainy Saturdays affect the duration of trips from the Loop to O'Hare airport.

![Hypothesis Testing](04_prueba_hipotesis.png)

## 5. Results and Conclusions
The analysis reveals patterns in taxi demand and highlights how data analysis can support transportation insights.

![Results](05_resultados.png)
